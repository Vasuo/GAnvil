# GAnvil

Платформа для создания, публикации и эксплуатации мультиплеерных 2д-экш видео-игр.

---

## **Функционал**

- **Акаунты:**
  - Есть функции регистрации и входа.
  - Сессия пользователя не сбрасывается при закрытии страницы.
  - Есть валидация форм, при ошибках данные не сбрасываются и выводятся оповещения.

- **Галерея:**
  - Страница с опубликованными играми, выглядит как пролистываемые слайды.
  - Каждый слайд отображает название, описание, плакат игры и кнопку для создания сесси по этой игре.
  - Есть различные кнопки перехода на другие страницы.
 
- **Мастерская:**
  - Страница с разрабатываемыми играми текущего пользователя, выглядит также как галерея.
  - Есть функции создания новой игры из базового шаблона или копирования существующей игры по названию.
  - Есть функции редактирования игры, изменения названия с описанием и публикации игры, кнопки располагаются на сладайх каждой игры.
  - Есть различные кнопки перехода на другие страницы.

- **Сессии:**
  - Страница с сессиями, выглядит как пролистываемые слайды.
  - Каждый слайд отображает название игры, описание, хоста и список игроков.
  - Для хоста отображается кнопки "начать", для игроков - кнопка "присоединится" если сессия планируется и "подключится" если уже идёт.
  - Есть различные кнопки перехода на другие страницы.

- **Корректное отображение игр:**
  - В галерее отображаются только опубликованные игры.
  - В мастерской отображаются только неопубликованные игры текущего пользователя.

- **Хост игры:**
  - Когда хост начинает сессию, у него запускается страница с js игрой.
  - Когда к сессии подключается игрок, хост добавляет его в список игроков.
  - Хост получает все инпуты от игроков через сервер, компилирует игру и передаёт результаты через сервер игрокам.
  - У хоста есть окно редактирования, в котором он может менять любые доступные параметры игры.
 
- **Страница игрока:**
  - Когда игрок подключается к сессии, у него запускается страница игрока, которая передаёт серверу инпуты игрока и выводит получаемые от сервера результаты на экран.
 
- **Режим редактирования:**
  - В ражиме редактирования мастерской открывается страница хоста игры, только к такой игре нельзя подключится, а все изменения введённые через окно редактирования можно сохранить в данные игры.

## **Технологии**

- **Frontend:**
  - HTML, CSS (Bootstrap 5).
  - JavaScript.

- **Backend:**
  - Python (Flask) - сервер.
  - Flask_socketio (с eventlet) - для быстрой передачи данных в игре.
  - SQLAlchemy для работы с базой данных.
  - Flask-Login и Flask_wtf для аутентификации пользователей и других форм.
  - Werkzeug - хэш-функция..

- **База данных:**
  - SQLite (для простоты разработки).
